# Spotify Clone

A full-stack Spotify Clone application built to recreate the core experience of a modern music streaming platform. This project includes music playback, playlist management, authentication, responsive UI, and dynamic music browsing features.

---

## Features

* User Authentication & Authorization
* Music Playback Controls
* Playlists & Album Sections
* Search Songs & Artists
* Responsive Spotify-Inspired UI
* Real-Time Music Player
* Dynamic Routing
* REST API Integration
* State Management

---

## Installation

Clone the repository:

```bash id="m2otz8"
git clone <your-repository-url>
```

Move into the project directory:

```bash id="3mjlwm"
cd spotify-clone
```

Install dependencies:

```bash id="j5d8m4"
npm install
```

Run the development server:

```bash id="90kgv0"
npm run dev
```

Open:

```text id="q3m7mi"
http://localhost:3000
```
